Learning Roadmap Explained

Why Need A Roadmap?

  • Because JavaScript is anywhere
  • Because we need a mental map that helps in:

    • -> being aware of what we know and what we still do not know

    -> being able to refresh what we already know

    -> being able to create an end-to-end applications
    with JavaScript-based technology stack
